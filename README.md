# 🎓 Student Record Management System

A Python-based console application designed to manage student records efficiently.
The system allows authenticated users to perform CRUD operations (Create, Read, Update, Delete) along with advanced search functionality and persistent data storage.

---

## 📖 Overview

This project demonstrates how core programming concepts such as Object-Oriented Programming, file handling, and data management can be used to build a complete working system.

The application is menu-driven and provides a structured interface for managing student information in a simple and efficient way.

---

## 🚀 Key Features

### 🔐 Authentication System

* User registration
* User login
* Access control before entering the system

### 📚 Student Record Management

* Add new student records
* Update existing records
* Delete records
* Display all stored records

### 🔍 Search Functionality

* Search by roll number
* Search by name
* Search by email
* Search by gender
* Search by branch
* Search by course
* Search by semester

### 💾 Data Persistence

* All data is stored using file handling (`pickle`)
* Data is automatically saved and loaded

## ▶️ How to Run

1. Make sure Python 3 is installed
2. Download or clone the project
3. Run the program:

```bash
python main.py
```

---

## ⚙️ Working Flow

1. User registers or logs in
2. System loads existing data from file
3. User selects operations from menu
4. Actions are performed (add, update, search, etc.)
5. Data is saved automatically after modifications
