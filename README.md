# 🎓 Student Record Management System

A **menu-driven console application** built using Python that allows users to manage student records efficiently with authentication and persistent storage.

---

## 🚀 Features

### 🔐 Authentication System

* User Registration
* User Login
* Secure access to system

### 📚 Student Record Management

* Add new student record
* Update existing record
* Delete student record
* View all records

### 🔍 Search Functionality

* Search by Roll Number
* Search by Name
* Search by Email
* Search by Gender
* Search by Branch
* Search by Course
* Search by Semester

### 💾 Data Persistence

* Data stored using `pickle`
* Automatic save & load
